Welcome

Novacta.Documentation.ShfbTools is a .NET library that enhances a Sandcastle Help File Builder installation in what follows.

  • Management of LaTeX equations in documentation files.

  • Support for inserting images in Sandcastle reference topics.

  • Support for adding or removing families of class names to be highlighted in code examples.

The project targets the .NET Framework, version 4.6.2.

The library is licensed under the MIT license.

Documentation

The current documentation includes the following topics.

  • Novacta.Documentation.ShfbTools release notes

  • Adding LaTeX equations includes topics showing how to define LaTeX content in documentation files.

  • Using images in reference topics describes how to show images in Sandcastle reference documents.

  • Highlighting class names walks you through the definition of class names to be highlighted in code examples.

  • Testing the tools instructs about the setup required to test the SHFB tools.

  • Library reference contains reference information about library types by namespace.

  • SampleClassLibrary Namespaces points to a sample documentation obtained through the Novacta.Documentation.ShfbTools library.
